ESPN4CC4C Admin

Quick links to health checks, EPG/M3U outputs, filters, “what’s on”, deep-links, and database maintenance.

Maintenance Admin

Database refresh / VACUUM and scheduler status.

  • /admin/refresh – refresh & VACUUM dashboard (manual + scheduled)
  • /health – simple service health check

EPG & Playlists Channels

Outputs you point Channels DVR or other clients at.

  • /out/epg.xml – XMLTV EPG
  • /out/playlist.m3u – standard M3U
  • /out/playlist.ch4c.m3u – experimental CH4C M3U

What’s On / Lanes

Current programming in each virtual channel lane.

  • /whatson_all?format=txt – all lanes (text)
  • /whatson_all?format=json – all lanes (JSON)
  • /whatson/{lane} – single lane (txt or json)

Filters & Setup

Inspect and tune your filters based on what’s in the DB.

  • /setupfilters – interactive filter helper
  • /filters – current filters (HTML)
  • /filters/json – current filters (JSON)

Deep-Link & Debug

ESPN deep-link helpers and resolver debug tools.

  • /deeplink/{lane} – sportscenter:// URL for a lane
  • /api/test/{lane} – deeplink test matrix
  • /vc/{lane} – main resolver endpoint
  • /vc/{lane}/debug – resolver debug JSON
Tip: start at /admin/refresh to confirm the scheduler is running and the database is healthy, then wire /out/epg.xml and /out/playlist.m3u into Channels DVR.
